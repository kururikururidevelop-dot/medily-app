# 単体テスト和様書（スクリーンテスト）

## 概要
本ドキュメントは、Medilyアプリの主要画面および機能の動作確認を行うためのテスト仕様書である。
LINE連携機能（通知、ログイン等）は環境依存のため対象外とする。

## テスト環境
- OS: Windows
- Browser: Automation Browser (Chrome equivalent)
- URL: http://localhost:3000 (Development Environment)
- Mock User: dev-mock-user

## テストケース一覧

### 1. ホーム画面 (`/home`)
| ID | テスト項目 | 手順 | 期待値 | 確認 |
|---|---|---|---|---|
| H-01 | 画面表示 | `/home` にアクセス | ヘッダー、サマリー、貢献度グラフ、タブが表示されること | [ ] |
| H-02 | タブ切り替え | 「質問」タブをクリック | 「質問」一覧が表示され、フィルタ（ステータス、カテゴリ）が表示されること | [ ] |
| H-03 | フィルタ操作 | カテゴリを1つ選択 | リストがフィルタリングされ、再描画されること | [ ] |
| H-04 | ページネーション | スクロールダウンし「もっと見る」をクリック（データがある場合） | 追加の質問がロードされ、リストの下に追加されること | [ ] |
| H-05 | 遷移確認 | 質問カードをクリック | 該当の質問詳細画面へ遷移すること | [ ] |
