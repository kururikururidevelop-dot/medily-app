# 医療科目ごとの具体的な質問者ペルソナ

# 内科（生活習慣病・風邪・長期服薬）
ペルソナ例：40代男性・会社員
最近健康診断で血糖値が高いと言われて不安
通院先の説明が短くてよく分からない
薬を変えたいが、今の医者に言いづらい
「他の人はどういう治療をしてる？」と知りたい
→ 不安解消・治療方針の比較・患者体験を知りたい層

# 整形外科（腰痛・ヘルニア・関節）
ペルソナ例：50代女性・パート
慢性的な腰痛で通院中
どこの整形外科が丁寧？リハビリの質は？
口コミはあるが、長期通院した人の声を探してる
→ 通院の「相性」やリハビリの実態を知りたい層

# 皮膚科（アトピー・ニキビ・アレルギー）
ペルソナ例：20代男性・大学生
ニキビ治療の効果が出ず、病院を変えるか迷ってる
薬の強さや副作用を経験者に聞きたい
→ 治療比較や成功例・失敗例を知りたい層

# 精神科・心療内科（うつ・不眠・不安症）
ペルソナ例：30代女性・会社員
初めて精神科を受けるので不安が大きい
どんな雰囲気？医者は話を聞いてくれる？
薬の副作用が不安
→ 病院選びの心理的ハードルが高い層
※ センシティブなので匿名性・非公開質問が重要

# 婦人科（更年期、PMS、不妊治療）
ペルソナ例：30代後半女性
不妊治療で何を選ぶべきか迷っている
専門クリニックの差を知りたい
男性の医師に相談しにくい話題
→ 同じ経験者（回答者）との非公開相談のニーズが強い

# 小児科（子供の病気・発達相談）
ペルソナ例：30代ママ
子供の湿疹やアレルギーで病院選びに困っている
発達相談で「どの病院が丁寧か」を知りたい
→ 信頼性・安心感を求める層

# 歯科（矯正・インプラント）
ペルソナ例：20代女性
矯正の値段の違いが分からない
実際の痛みや通院頻度を聞きたい
→ 経験者レビューの価値が高い分野

# 眼科（白内障・レーシック）
ペルソナ例：60代男性
手術の体験談が知りたい
術後の生活がどう変わったか気になる
→ 経験談への需要が大きい科目

# がん診療（セカンドオピニオン）
ペルソナ例：50代男性・家族の付き添い
セカンドオピニオン先を探している
大病院の実際を知りたい
→ 高度医療で患者経験に最も価値が出る領域
※ ただし個別医療アドバイスは禁止・体験談レベルに寄せる必要あり

# 総合的な質問者像（まとめ）
**共通する動機**
病院や医師の「実際の対応」を知りたい
通院前の不安を減らしたい
他の患者がどう感じたか知りたい
セカンドオピニオンを検討中
長期通院のストレス・不満がある
