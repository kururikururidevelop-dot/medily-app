| キャッチコピー             | テキスト | □    | Heading H2         |
| サービス説明テキスト       | テキスト | □    | BodyText           |
| サービスイメージ           | 画像     | □    | HeroImage          |
| 質問事例                   | 表示領域 | □    | QuestionCardList   |
| 質問してみるボタン         | ボタン   | ●    | PrimaryButton      |

| フッターテキスト           | テキスト | □    | FooterText         |

### 画面仕様補足（設計レベル概要）
* 本画面は**未ログインユーザー向けの集客・導線専用画面**とする
* 「質問事例」に公開質問の一部と「あなたの経験が必要です」と回答者に訴求するメッセージを表示する
* 「質問してみるボタン」押下時は以下のいずれかに遷移する
  * 未ログイン時：OAuth選択画面（A020）
  * ログイン済み：質問投稿画面（B010）
* 「回答してみるボタン」押下時は「質問一覧（公開）（E011）」に遷移する
* Heroエリアにはサービスの世界観を伝えるメインビジュアルを表示する
* フッターは全ページ共通のため、Footerコンポーネントを流用する

### 使用コンポーネント

* Heading
* BodyText
* PrimaryButton
* OAuthButton
* Footer

### 画面項目定義

| 項目名                     | 種別     | 区分 | 使用コンポーネント |
| -------------------------- | -------- | ---- | ------------------ |
| サービスロゴ               | 画像     | □    | LogoImage          |
| 画面タイトル               | テキスト | □    | Heading H1         |
| 説明テキスト               | テキスト | □    | BodyText           |
| LINEログインボタン         | ボタン   | ●    | OAuthButton (LINE) |
| 利用規約リンク             | リンク   | □    | TextLink           |
| プライバシーポリシーリンク | リンク   | □    | TextLink           |
| フッターテキスト           | テキスト | □    | FooterText         |

### 画面仕様補足（設計レベル概要）
* 本画面は**認証方式選択専用画面**とする
* ユーザーは以下の方法で認証できる
  * LINEアカウントによる認証（LINE Messaging API との連携）
* 各ボタン押下時は外部OAuth認証画面へ遷移する
* 認証成功後の遷移先は以下ルールで制御する
  * 初回ログイン：初回プロフィール登録画面（A030）
  * 登録済みユーザー：ホーム画面（A040）
* 利用規約・プライバシーポリシーは常に確認可能とする

  * 回答対象としてマッチングされた質問
* ステータスは視覚的に区別できるようバッジ表示とする
* 通知アイコンは未読がある場合に強調表示される

## L010｜質問投稿（LIFF）
### 使用コンポーネント
* Heading
* BodyText
* TextArea
* TextInput
* SelectBox
* RadioButtonGroup
* Checkbox
* PrimaryButton
* SecondaryButton

### 画面項目定義

| 項目名           | 種別         | 区分 | 使用コンポーネント |
| ---------------- | ------------ | ---- | ------------------ |
| 画面タイトル     | テキスト     | □    | Heading H1         |
| 質問タイプ選択   | ラジオ       | ●    | RadioButtonGroup   |
| 質問内容         | テキスト入力 | ●    | TextArea           |
| テンプレート選択 | セレクト     | ○    | SelectBox          |
| カテゴリ選択     | セレクト     | ○    | SelectBox          |
| 回答選択肢       | テキスト入力 | ○    | TextInput (複数)   |
| 地域             | セレクト     | ●    | SelectBox          |
| 性別フィルタ     | ラジオ       | ○    | RadioButtonGroup   |
| 年齢フィルタ     | セレクト     | ○    | SelectBox          |
| 位置情報の利用   | チェック     | ○    | Checkbox           |
| URL添付の許可    | チェック     | ○    | Checkbox           |
| 公開レベル       | ラジオ       | ●    | RadioButtonGroup   |
| 投稿確認ボタン   | ボタン       | ●    | PrimaryButton      |
| キャンセルボタン | ボタン       | ○    | SecondaryButton    |

### 画面仕様補足（設計レベル概要）
* LIFF 内で動作する**質問投稿専用フォーム画面**
* 質問タイプは以下の2つから選択必須：
  * **新規質問**：新しい相談を投稿（マッチング対象全回答者へマルチキャスト）
  * **追加質問**：過去の質問の続きとして投稿（過去質問でマッチングした全回答者へマルチキャスト）
* テンプレート選択時は質問文・回答選択肢が自動入力
* 地域は必須項目で、プロフィール情報から自動設定
* 投稿確認ボタン押下で L011 へ遷移
* キャンセルボタンでホーム（A040）へ戻る
